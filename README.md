# 📊 Insurance Data Dashboard (Power BI Project)

This project is an end-to-end Data Analyst solution built using Power BI and DAX to analyze insurance data and generate actionable business insights. The dashboard helps in understanding customer behavior, policy distribution, claims performance, and premium trends.

 # Tools & Technologies:
 Power BI
 DAX (Data Analysis Expressions)
 Microsoft Excel
 
 ## 📌 Key Features
 KPI Tracking: Total Policies, Customers, Claims, Expiring Policies
 Customer Segmentation: Age group and gender analysis
 Claims Analysis: Approved vs Pending vs Rejected claims
 Payment Status Insights
 Policy Type Distribution
 Premium Growth Trend Analysis

## 📈 Business Insights
 Higher number of policies observed in senior age groups (65+)
 Claims data shows a significant portion in pending status
 Balanced distribution across genders
 Premium trends indicate fluctuations over time
